# CobWebX

CobWebX is a lightweight plugin that controls temporary block placement inside WorldGuard regions.

## Features

- Region based block control
- Automatic block removal
- Group system
- Debug mode
- Multiverse compatible

## Commands

/cobwebx help  
/cobwebx reload  
/cobwebx check
